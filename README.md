# MySQL E-Commerce Project

This project demonstrates a structured e-commerce database using MySQL.

## Features
- Database and Table Creation
- Data Insertion
- JOIN Queries
- Aggregate Functions
- Revenue Calculation
- Update and Delete Operations

## Tools Used
- MySQL Workbench

## Database Structure
- Customers Table
- Products Table
- Orders Table
